## 🚀 Sobre o Projeto

Excelia é uma plataforma de IA especializada em Excel que ajuda analistas financeiros, controllers e profissionais a:

- ✅ Corrigir fórmulas automaticamente
- ✅ Gerar dashboards executivos
- ✅ Automatizar tarefas com VBA
- ✅ Auditar planilhas em segundos
- ✅ Aprender Excel avançado

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/Artemixian/Excelia.git
```

2. Abra qualquer arquivo `.html` no navegador:
```bash
# Recomendado: versão principal
open index.html

# Ou versão oficial completa
open excelia-landing-official.html
```

## 📊 Funcionalidades

### Correção de Fórmulas
- Detecta e corrige #REF!, #VALUE!, #N/A
- Identifica referências circulares
- Sugere otimizações

### Geração de Dashboards
- KPIs automáticos
- Gráficos interativos
- Layouts profissionais

### Automação VBA
- Gera macros a partir de descrições
- Scripts personalizados
- Automação de tarefas repetitivas

### Auditoria Inteligente
- Score de integridade (0-100)
- Detecção de anomalias
- Relatórios detalhados
